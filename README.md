# AI Books Analyze

**Live Project:** [https://leyouhong.xyz/](https://leyouhong.xyz/)

AI Books Analyze is a full-stack web application for managing, reading, and analyzing books with the help of AI. It features a Django REST backend and a React (Vite) frontend.

## Features

- Browse and manage a personal library of books
- Upload PDFs and extract metadata automatically
- AI-powered book analysis (summaries, themes, takeaways, and more)
- AI chatbot to discuss and explore books
- Reading progress tracking and personal bookshelves
- Reviews, tags, and tag-following with notifications
- User accounts with email verification

## Project Structure

- `backend/` — Django REST API
- `frontend/` — React + Vite single-page application

For more details, see the README files inside the `backend/` and `frontend/` directories.
